## Why
The project needs a stable Vite + React + TypeScript baseline before feature work can start. A consistent foundation reduces rework and enables all following changes.

## What Changes
- Initialize and standardize Vite React TypeScript setup for admin template work.
- Configure Tailwind CSS v4, strict TypeScript, path alias `@/*`, and base lint/format scripts.
- Define global style and theme token foundation used by later UI changes.

## Capabilities

### New Capabilities
- `project-foundation`: Base frontend architecture and tooling conventions for the template.

### Modified Capabilities
- None.

## Impact
- Affects project bootstrap, configuration files, and base style system.
- Unblocks all subsequent OpenSpec changes.
