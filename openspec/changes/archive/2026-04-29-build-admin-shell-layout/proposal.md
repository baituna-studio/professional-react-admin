## Why
The admin shell is the backbone of navigation and responsive usability. Without a defined shell, feature pages cannot deliver a coherent app experience.

## What Changes
- Build responsive app shell with desktop sidebar, mobile drawer, and topbar.
- Add route-aware navigation and active menu styling.
- Include topbar controls for search trigger, notifications entry, theme toggle, and user menu.

## Capabilities

### New Capabilities
- `admin-app-shell`: Responsive navigation shell for authenticated areas.

### Modified Capabilities
- None.

## Impact
- Affects `components/layout` and shared navigation behavior.
- Becomes the structural container for all protected routes.
